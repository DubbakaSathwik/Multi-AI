System Core Version: v2.5.0 Premium Interface Engine: HTML5 / CSS3 / Vanilla Javascript Active Framework: AI OS Compare Core
About Nexora AI

One workspace for comparing AI answers

Nexora AI is a single-page comparison workspace. You write one prompt, send it to multiple AI models, compare the answers, continue with follow-up queries, and save useful sessions for later.

Parallel Answers

Use the left model icons to turn models on or off. The send button broadcasts your prompt to every active card.

Response Card Tools

Copy saves text to clipboard, Query asks a follow-up using the last answer, Download exports the card, Compare checks differences, and thumbs record feedback.

Best Response

The star button asks the judge to choose the strongest usable answer. Failed, unavailable, or rate-limited responses are ignored.

Personas & Prompt Roles

The Expertise page lets you pick a role, edit custom personas, or auto-generate a new assistant profile from a short idea.

Debate Arena

Choose two models, a topic, and round count. Nexora AI runs the debate and keeps a toggleable summary verdict.

Insights

Insights shows chat count, favorite model, site time, model wait time, likes/dislikes, and speed diagnostics.

Settings

Manage API keys, best-response preferences, per-model generation parameters, and workspace card rearrangement.

Profile & History

Profile stores local user details. History lets you rename, pin, delete, and reload saved chat sessions.

⭐ Best Response Personalization

Customize the evaluation algorithm by choosing your preferred answer criteria. Clicking "Select Best Response" will highlight the card that scores highest according to your selected preference.

⚙️ Custom API Key Settings (BYOK)

Enable custom API keys to use your own credentials. Key values are stored locally in your browser session.
